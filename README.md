# django-study

### 환경
- Python 3.11
- IDE: Visual Studio Code
- Framework: Django
- Database: MySQL


### 설명
- Django를 기반으로 게시판 CRUD 구현
- MVT 패턴을 활용하여 애플리케이션의 구조 정리
  - 각각의 역할을 수행하는 Model, View, Template로 코드 분리
- Django의 ORM을 이용하여 데이터베이스와의 상호작용 처리
  - Python 코드를 사용하여 데이터 조작을 수행하며, SQL 쿼리 작성에 대한 번거로움 ⬇
